Step 3B
Is child born between Sept 1 - Dec 1, 2023?
if YES
⚠️ Assessment Available for FS1
Schools assess readiness in consultation with parents.
If ready: FS1 this year
If not ready: FS1 next year
Step 3C
Is child born between Sept 1 - Dec 1, 2024 (or 2025, 2026...)?
if YES
⚠️ Annual Assessment Policy
If turning 3 between Sept-Dec of school year:
Schools assess readiness in consultation with parents.
If ready: FS1 that year
If not ready: FS1 following year
Applies annually from 2026/2027 onwards
IF NO
NO BRITISH CURRICULUM

December 31 cut-off (child must turn 3 before Jan 1)

  • September / December 2022 → FS2/KG1
